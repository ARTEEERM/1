# GitHub User Finder

**Автор:** Иванов Иван

## Описание

GitHub User Finder — это GUI‑приложение на Python для поиска пользователей GitHub с использованием официального API. Программа позволяет:


* искать пользователей GitHub по имени;
* просматривать основную информацию о пользователях;
* добавлять пользователей в избранное;
* сохранять избранных пользователей локально в формате JSON;
* управлять списком избранных пользователей.

## Технологии

* Python 3
* tkinter — для создания графического интерфейса;
* requests — для работы с GitHub API;
* JSON — для локального хранения избранных пользователей.
